Return Merchandise Authorization (RMA) Management - Odoo App

This app allow your customer(s) to create return RMA order and management of Return Merchandise Authorization (RMA). And also allow your customer to view return order(s) from My Account Portal.


Main Features:
  • Customer can create RMA return order by logged into Portal / My Account.
  • Customer can go to order and click "Return" button to return item which will generate return order (RMA Order) in backend.
  • Customer can see list of RMA orders in Portal / My Account.
  • Allow your customer to specify return reason.
  • We have added workflow on return order in backend (Draft->Confirm->Approve->Return) Allow your stock team to do return pickings from delivery orders. (Odoo Standard Feature) It allow only single product/item on RMA order so if you want to allow your customer to request for multiple sales order line return case and then want to return RMA in backend with multiple items/products then it will need multiple RMA. If you want mulitple lines link on single RMA Order (Return Order) you can check our app https://apps.odoo.com/apps/modules/13.0/website_request_return_rma_odoo/
  • Link up of RMA order with Sales order.
  • On customer form smart button showing all return RMA order related to that customer.
  • Print RMA PDF Report - Print Return Report.
  • For more details please watch Video and see below screenshots.
